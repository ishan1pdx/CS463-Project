# Personal Portfolio Website

## Overview

This project is a personal portfolio website created for CS 463 Intro to Web Development. The website highlights my education, work experience, technical skills, and software development projects.

## Features

* About section
* Experience section
* Projects section
* Contact form
* Responsive design
* JavaScript interactivity
* Bootstrap accordion component

## Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap

## Projects Featured

### GRE Vocabulary Chrome Extension

Chrome extension that uses spaced repetition to help users learn vocabulary words.

Repository:
https://github.com/ishandhar3404/GRE-Vocab-Teacher

### NBA Player Comparison Tool

Web application that compares NBA player statistics and generates visualizations.

Repository:
https://github.com/ishandhar3404/nba-analyzer

### Spotify AI Song Recommender

Application that creates playlists using machine learning and Spotify audio features.

Repository:
https://github.com/ishandhar3404/Spotify-AI-Song-Recommender

## Running the Project

1. Clone the repository.
2. Open the project folder.
3. Open index.html in a web browser.
